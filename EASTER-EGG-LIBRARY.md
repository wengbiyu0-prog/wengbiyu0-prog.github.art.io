# EDIMAGE WORLD 彩蛋库

版本：v0.1  
用途：储存 EDIMAGE WORLD 的非常规触发词与恶作剧情节。  
规则：彩蛋只在进入互动世界之后展露，不在 Home 页、抽卡页或体量页提前暴露。

## 彩蛋触发总则

1. 用户输入的 idea 命中彩蛋词汇时，系统记录彩蛋状态。
2. 彩蛋不会改变抽卡流程。
3. 彩蛋只在互动小说世界中插入。
4. 彩蛋情节可以与原叙事毫无关联。
5. 彩蛋可以打破第四面墙。
6. 用户无论选择哪个彩蛋选项，彩蛋结束后都必须回归原本叙事。
7. 彩蛋不应污染主线世界状态，除非后续明确设定为“可污染彩蛋”。

## 彩蛋 001

状态：active  
触发词：蝴蝶  
触发阶段：进入互动世界后  
插入方式：在常规互动情节之间突然插入非常规决策树  
回归规则：用户完成任一彩蛋选择后，回到原来的文本叙事，并接续彩蛋插入前的主线状态。

### 触发情节

蝴蝶大战金刚侠，金刚侠惨败，被迫上梁山，尔后号召108好汉，在水浒边上起义，史称爱尔兰史上最伟大的革命。

### 彩蛋选项方向

1. 支持蝴蝶继续追击金刚侠。
2. 劝金刚侠立刻上梁山。
3. 要求 108 好汉解释为什么这件事发生在爱尔兰史上。
4. 拒绝理解这一切，并要求世界回到原来的小径。

### 回归文案方向

彩蛋结束后，系统应使用一句短句把用户带回主线，例如：

`这场革命像一张被错误塞进放映机的胶片，闪了一下，随即烧毁。你重新听见原本那条小径上的声音。`

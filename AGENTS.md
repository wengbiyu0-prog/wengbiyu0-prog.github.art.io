# EDIMAGE WORLD Agent Guide

This repo contains the EDIMAGE WORLD interactive fiction website: a Node-backed, art-oriented text world with AI generation, invite-code access, visual skins, a local knowledge base, and an easter-egg library.

## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues for `wengbiyu0-prog/website-edimage`. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default five-label triage vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, and `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repo: read `CONTEXT.md` and root `docs/adr/` if they exist. See `docs/agents/domain.md`.
